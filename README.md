## 📦 Installation

```bash
# Cloner le repository
git clone https://github.com/digiservweb/website.git
cd website

# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev

# Build pour production
npm run build

# Prévisualiser le build
npm run preview
```
